Простое приложение для учета товара на складе.
 
## Как запустить:
 пока просто в разработке.

 ### Что мы храним в базе данных
 1. Товары(products) с полями: название , id , цена и количство на складе,ссылка на котегорию(внешний ключ), дата добавления!
 2.  Категории товаров: название, id, описание категории
